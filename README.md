This is a calculator for your BMI, also known as Body Mass Index.
To run this calculator, you need to create a folder containing all my files.
Once you have done that, open the HTML file "BMI_Calculator.html" and run it in a browser like Microsoft Edge or Google Chrome. 
This will create your BMI calculator for you to test your BMI.
After this, you may enter your age, weight and height into the inputs, for the formula to calculate the BMI.
To understand what BMI is, you can navigate through the header to the "What's BMI" and "Average BMI" links.
